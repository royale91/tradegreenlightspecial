Indicator 02 · Exit Pane

GLS OSOB — The Exit Companion

A separate pane below price. Its one job is telling you when to get out and scale — never when to get in.

How to Read It

The OSOB Pane

The pane reads how stretched momentum is, so you bank profit at the extremes instead of round-tripping a winner.

  • ⚪ White wave drifting — let it ride
  • 🔴 Red top (overbought) — long getting tired → scale
  • 🟢 Green bottom (oversold) — short getting tired → scale
  • 🟡 GOLD dots = the EXIT BELL — extreme rolling over → take it / tighten
  • ▲▼ zero-cross — momentum flipped
⚠ Honest Framing

Exits Only — Never Entries

Overbought/oversold oscillators work in ranges and fail in trends — so OSOB is for EXITS and scaling only, never to fade a trend or pick an entry. That's why it's a companion to GLS, not a standalone signal.

Pair it with the Yellow Man warning candle on the main chart: when the Yellow Man lights up and OSOB rings the exit bell, the move is done — protect the profit and wait for the next clean setup.
